CSS-Animations

CSS-də animasiya xüsusiyyəti; Rəng, fon rəngi, hündürlük və ya genişlik kimi bir çox CSS xüsusiyyətlərini canlandırmaq və təsir etmək üçün istifadə edilə bilər. Animasiyalar elementə tədricən bir formadan digərinə keçməyə imkan verir. CSS animasiyasından istifadə etmək üçün əvvəlcə bəzi əsas qaydaları və parametrləri təyin etməlisiniz. Qaydalar və parametrlər müəyyən vaxtlarda elementin hansı formaya malik olacağını müəyyənləşdirir.
CSS Animasiyalarından istifadə etmək üçün @keyframes qaydasından istifadə etməliyik. @keyframes qaydası animasiyanın əsas strukturunu yaradır. Animasiyanın müəyyən vaxtlarda hansı formada dəyişəcəyini müəyyən edən qaydadır.

Color change with Animations

Keyframe-in iş məntiqini daha yaxşı başa düşmək üçün rəng keçidi nümunəsi edək.Stil teqləri arasında keyframe-I tətbiq edək və onu rengli animasiya adlandıraq. From-da yazılan hissə ilk formada görünəndir , To-da olan isə animasiyadan sonra tətbiq ediləcək formadır.

Animasiyanın Faizlərə Bölünməsi

Key Frame-də animasiya dəyərini faiz olaraq təyin etməklə də rəngi dəyişə bilərik. Məsələn, animasiya rənglərini 4 ayrı bölməyə ayıraq. Animasiya prosesində; 10% tamamlandıqda qırmızı rəng, 40% tamamlandıqda yaşıl rəng, Animasiya 60% tamamlandıqda mavi, 100% tamamlandıqda isə sarı rənglə bitəcək.

Animasiya ilə Hərəkətləndirmə

Key Frame-lə biz nəinki rəngləndirə, həm də obyektlərin yerini dəyişdirə, hərəkət effekti və hərəkət animasiyası verə bilərik. Məsələn, div-i aşağı hərəkət etdirmək istəyiriksə, position teqlərindən istifadə etməliyik.

Animasiyanın Keçidlərini nezerden kecirek

Animasiyada tətbiq olunan keçid effektlərinin növlərini dəyişmək mümkündür. Keçid effektlərinin növünü animation-timing-function etiketi ilə edirik.

  • animation-timing-function: linear; => Fasiləsiz xətti keçidi təmin edir.
  • animation-timing-function: ease; => Fasilə verərək addım-addım keçidlər edir.
  • animation-timing-function: ease-in-out; => Yavaşdan sürətliyə keçir.
  • animation-timing-function: ease-out; => Yavaş başlayır, tez bitir.
  • animation-timing-function: ease-in; => Sürətlə başlayır, yavaş bitir.

Animasiyanın Yönünü Təyin etmək

Bunun üçün biz animation-direction-dan istifadə edirik

normal Cari dəyər. Animasiya normal olaraq göstərilir (irəli)
reverse Animasiya tərs istiqamətdə göstərilir (geri)
alternate Animasiya əvvəlcə irəli, sonra geriyə doğru göstərilir
alternate-reverse Animasiya əvvəlcə geriyə, sonra irəliyə doğru oynanılır

CSS-Transitions

CSS transition Özelliği bir elementə hover olduğunda ya da .active kimi bir classname aldğında dəyişən stil özelliklərini keçişli bir şəkildə verməyə köməklik edir.Aşağıdakı property-ləri var

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
  • transition kısa yolu yazımı

Transition Duration

Keçid müddəti vaxtı bildirmək üçün istifadə edilir. Keçid vaxtını saniyə və millisaniyələrlə daxil etməlisiniz.

Transition-timing-function

Tətbiq olunan keçid effektlərinin növlərini dəyişmək mümkündür

ease animasiya ortaya doğru sürətlənir və sonra dayanana yaxın yavaşlayır.
ease-in animasiya yavaş başlayır və tamamlanana qədər sürətlənir.
ease-out Animasiya tez başlayır və yavaş-yavaş bitir.
ease-in-out Animasiya yavaş-yavaş başlayır və sürətləndirmək üçün keçid edir, sonra animasiyanı tamamlamaq üçün yenidən yavaşlayır.
linear Animasiya sürəti animasiyanın əvvəlindən sonuna qədər bərabər şəkildə davam edir.
